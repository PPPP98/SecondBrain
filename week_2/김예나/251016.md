# 아이디어 구체화 - 장기 기억 노트(세컨드 브레인)

## 제안 -> 기능 추가

> 데일리 리뷰 알림 시스템

자동화된 알림 시스템, 단순 리마인더가 아닌 자체로 하나의 학습 경험 제공

1. 무엇을 어떻게?

- 망각 곡선 기반 알고리즘 : ‘오늘 꼭 복습해야 할’ 최적의 노트 선정
- 사용자 설정 기반 : 사용자가 원하는 시간 혹은 채널(이메일, 푸시 알림) 직접 설정 가능

2.  핵심 기능

    - 2.1. 적응형 퀴즈 시스템 연동

      - 노트 내용을 이메일/알림에 간략히 보여줌
      - 해당 노트 기반으로 생성된 퀴즈를 알림/이메일 내용에 포함
      - 사용자는 앱을 켜지 않고도 바로 퀴즈를 풀 수 있고 ‘능동적 회상’ 가능

    - 2.2. 학습 스트릭

      - 스트릭 유지 푸시 알림 전송
      - "37일 연속 학습을 이어갈 시간이에요!"

    - 2.3. 팟캐스트 활용

      - "바쁘신가요? 출근길에 ‘오늘의 복습 노트’ 팟캐스트(7분)로 먼저 들어보세요."

## 아이디어 고도화

회의 및 팀미팅

- 타겟층 설정
  - `학습 내용을 완전히 암기해서 온전히 머릿속에 넣고 싶은 건지?`
  - `(머리에 넣은 것처럼) 내가 필요할 때 바로 꺼내 쓰고 싶은 건지?`
- 서비스 명 : **"세컨드 브레인"**
- 서비스 동작 흐름
  - first brain : 사용자
  - second brain : 우리 서비스 DB
  - 동작 시나리오
    - 크롬에서 'ai' 검색
    - second brain 과거 정리 노트·학습 기록 자동으로 띄워줌
    - 브라우저에 개인화 결과 제시
