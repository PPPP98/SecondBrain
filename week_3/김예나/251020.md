# 아이디어 고도화

- 일상 속에서 '적극적으로' 학습/리마인드에 도움을 줄 수 있는 게 없을까 고민해봄

### 타겟

1. 기록을 잘 하고 있는 학습자
2. 기록이 귀찮은 학습자
3. 일반 이용자(학습자x)

## 서비스 흐름

### 1. 저장

- LLM 채팅 자동 저장(익스텐션)
- 영상 요약 저장
- 블로그 글 자동 저장

* 캘린더, 건강, 녹음 데이터

### 2. 가공

- 마크다운 형식 정리
- 유사도 기반 자동 태깅
- 카테고리화

* 개인 clone(나와의 대화)

### 3. 반응

- 학습 -> 리마인드/퀴즈
