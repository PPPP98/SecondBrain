## 코드 리뷰 자동화 시스템

- 셀프 호스팅을 통해 n8n 워크플로우 생성 완료.

```yaml
services:
  nginx:
    image: public.ecr.aws/nginx/nginx:1.27-alpine
    container_name: nginx
    restart: unless-stopped
    network_mode: host
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx.conf:/etc/nginx/nginx.conf:ro
      - /etc/letsencrypt:/etc/letsencrypt:ro
    mem_limit: 128m
    memswap_limit: 128m
    cpus: "0.1"

  n8n:
    image: ghcr.io/n8n-io/n8n:latest
    container_name: n8n
    restart: unless-stopped
    network_mode: host
    volumes:
      - n8n_data:/home/node/.n8n
    environment:
      - WEBHOOK_URL=https://e107.kro.kr/
      - N8N_PROTOCOL=https
      - N8N_HOST=e107.kro.kr
      - GENERIC_TIMEZONE=Asia/Seoul
      - N8N_REDIS_HOST=localhost

volumes:
  n8n_data: 
    external: true
```

- certbot을 활용하여 letsencrypt ssl 인증서 발급 후 도메인, n8n 띄우기 완료
