TIL: Second Brain 서비스 아이디어 고도화 고민 과정 (이석재 개인적 의견)

2025.10.20

초기 상황
노션 기반 LLM 학습 관리 서비스를 고민 중

타겟: 디지털 학습 미숙, 정리 못하는 대학생
구상 기능: 망각곡선 기반 복습, 학습 로드맵 추천, 캘린더/워치 연동 등
문제: 여러 기능은 있으나 명확한 메인 기능이 없음


고민 1: 생체 데이터 활용 방향
    - 아이디어: 인지 에너지 기반 적응형 학습
    - 수면 패턴, 스트레스 지수 분석 → 현재 뇌 상태에 맞는 학습 콘텐츠 추천

    - "지금 집중력 85%" → 어려운 개념 추천
    - "피로도 높음" → 가벼운 복습 추천


- 우려되는 점

콜드 스타트 문제 심각 (데이터 축적 필수)
첫 사용자가 즉시 가치를 느끼기 어려움
기능 작동까지 시간 소요


고민 2: 진입 장벽 낮추기
    - 아이디어: 자동 정리 중심 접근
    - "필기만 하면 AI가 자동 정리" 콘셉

    - 자동 분류/태깅/연결
    - 망각곡선 기반 복습 알림
    - 관련 노트 자동 링크

- 한계점

"Second Brain"이라는 강력한 네이밍에 비해 기능이 약함
기존 노트 앱과의 차별화 부족
편의성 개선 수준에 그침



고민 3: 네이밍과 기능의 정합성
     "Second Brain"이라는 이름이 주는 기대치

    - 단순 정리 도구를 넘어선 무언가
    - "사용만 하면 뇌에 꽂아넣어주는" 느낌
    - 진짜 두 번째 뇌처럼 작동하는 시스템


이를 구현하려면?
→ 뇌과학 기반 접근 필요

고민 4: 뇌 작동 방식 모방 가능성
    - 아이디어: 뇌의 정보 처리 과정 구현
    - 뇌가 정보를 기억하는 방식:

    - 다중 감각 활용 (시각, 청각, 연상)
    - 반복과 인출 연습
    - 수면 중 기억 정리
    - 기존 지식과의 연결

이를 디지털로 구현한다면?

탐색한 기능 아이디어들
1. 기억 주입 모드 (콜드 스타트 해결)
45초 프로세스 가능성 검토:

핵심 추출 → 시각화 → 연상 비유 → 감각 자극 → 즉시 테스트
첫 사용부터 "외워진다" 체감 가능?
뇌과학적 근거 확인 필요

2. 신경망 매핑

개념 간 연결 시각화
취약한 연결 발견 및 학습 경로 제안
구현 복잡도 vs 효과 검증 필요

3. 수면 학습 연동

취침 전 요약, 렘수면 타이밍 활용
워치 API 연동 가능성 확인 필요
실제 효과에 대한 과학적 검증 필요

4. 인지 부하 관리

복잡한 내용 자동 분해
뇌의 작업 기억 한계 고려
