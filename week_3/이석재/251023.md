## 2025 10 24 TIL, 자율 PJT 방향성 확정

1. 프로젝트 개요
- Second Brain

서비스 한 줄 요약: 당신이 잊기 전에, AI가 먼저 챙겨주는 선제적 학습 관리 시스템

2. 핵심 기능 정리
자동 수집: 크롬 확장 프로그램으로 웹 페이지, 하이라이트, LLM 대화 내용 저장. (노트 묶기 포함)

AI 자동 정리: 노트 자동 요약, 키워드 추출, 개념 간 연결망(지식 그래프) 자동 생성

스마트 검색: 벡터 유사도 검색 및 RAG 기반 AI 챗봇 (노트 내용 질의응답)

선제적 리마인드 (핵심): 개인별 망각곡선 패턴을 학습하여 '잊기 전에' 복습 알림을 선제적으로 제공

3. 기존 서비스와의 차별점
기존 도구(NotebookLM 등): 사용자가 '필요할 때 찾아가는' 수동적 도구였지만 !

Second Brain은 AI가 사용자의 망각 패턴을 분석해 '필요하기 전에 먼저 찾아오는' 능동적인 파트너 (핵심은 선제적인 정보 제공, 최대한 뇌와 유사한 기능)

지식 그래프(Neo4j), 벡터DB(ChromaDB), 정형DB(MySQL)를 모두 활용하여 다각적인 데이터 연결 및 검색 제공

4. 백엔드(BE) 기술 스택
Spring Boot:

메인 API 서버

사용자 인증/인가 (Spring Security + JWT)

노트 CRUD, 데이터 관리 (JPA, Neo4j)

알림 스케줄링 (Spring Scheduler)

FastAPI:

AI 기능 전담 (요약, 키워드 추출, RAG 챗봇)

벡터 DB(ChromaDB) 연동

구성은 ->

Spring Cloud Gateway (API 엔트리 포인트 단일화 및 라우팅)

Spring Cloud OpenFeign (Spring <-> FastAPI 간 HTTP 통신)

DB :

PostgreSQL (사용자 정보, 노트 메타데이터 등 정형 데이터).

Neo4j (지식 그래프, 개념 연결망 시각화용).

Milvus (벡터 유사도 검색용)

Redis

5. 백엔드 - Spring (내 파트)

Spring Boot 기반 메인 API 서버 개발

Spring Data JPA 및 Spring Data Neo4j (Neo4j)를 이용한 DB 연동 및 관리.

Spring Security + JWT 기반 사용자 인증 및 API 보안 구현

'선제적 리마인드' 기능 구현을 위한 Spring Scheduler 설계

Spring Cloud Gateway, OpenFeign을 활용한 MSA 아키텍처 구축
